## Установка
* Установка зависимостей
   ```sh
   npm install
   ```
* Запуск приложения
   ```sh
   npm start
   ```
## Тестирование
* Тестирование бизнес-логики приложения
   ```sh
   npm test
   ```
* Тестирование функционала с использованием Cypress
   ```sh
   npx cypress open
   ```
